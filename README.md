# Satellite orbital dynamics modelization

Project Status: Completed (Academic Project)
Development Period: Apr 2024 - Jun 2024
Language: C

## Description

This is a standalone C implementation of a numerical solver that models the orbital motion of a satellite in a central gravitational field. The code integrates the equations of motion using a fourth-order Rungeâ€“Kutta method and allows the analysis of orbital evolution, energy conservation, and dissipative effects.
